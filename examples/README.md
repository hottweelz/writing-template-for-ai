# Examples

These are **illustrative samples** for a throwaway demo premise — *The Lighthouse Keeper's Daughter*, a made-up literary novel used only to show what the template produces. They are not part of any real book and not the novel this template was built from.

- [`SAMPLE_BLUEPRINT.md`](SAMPLE_BLUEPRINT.md) — an abridged Stage 1 blueprint (a few sections filled in) so you can see the shape and depth of what the AI generates from your brief.
- [`sample-chapter.md`](sample-chapter.md) — a short opening drafted to the template's prose standards, showing the chapter file format and the kind of writing the protocols aim for.

When you run the real flow, your `BOOK_BLUEPRINT.md` and `manuscript/chapters/` files will be far more complete. This folder exists so you can judge quality before you invest your own ten minutes.
