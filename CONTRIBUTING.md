# Contributing

Thanks for helping improve the AI Book Writing Template.

This repository is meant to stay reusable for many authors, genres, and AI coding tools. Contributions should improve the template itself, not add private manuscript content from a specific book project.

## Good contributions

- Clearer setup instructions for authors.
- Better AI handoff, memory, or agent-selection rules.
- Stronger editorial protocols or quality gates.
- Bug fixes in file paths, scaffold structure, or documentation.
- Small example placeholders that make the template easier to understand.

## Please avoid

- Personal manuscript drafts, private notes, or identifying source material.
- Tool-specific rules that break compatibility with Claude Code, Codex, Gemini CLI, or GitHub Copilot.
- Large rewrites without an issue or discussion first.
- Generated build outputs such as EPUB, PDF, MOBI, or temporary review artifacts.

## Workflow

1. Fork the repository.
2. Create a branch with a focused name, such as `docs/start-here-clarity`.
3. Make the smallest useful change.
4. Check that the README and AGENTS instructions still agree.
5. Open a pull request explaining what changed and why it helps authors.

## Template boundaries

Keep durable reusable guidance in template files such as `AGENTS.md`, `.ai/rules/`, `.ai/agents/`, and `docs/protocols/`.

Keep project-specific book facts out of template contributions unless they are clearly marked as placeholders.
